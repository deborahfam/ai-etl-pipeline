# Open datasets (downloaded, not committed)

CSV files here are fetched on demand so the repo stays small and sources stay traceable.

```bash
python3 data/fetch_open_datasets.py
```

| File | Source |
|------|--------|
| `titanic.csv` | [Data Science Dojo — Titanic](https://github.com/datasciencedojo/datasets) (public domain–style teaching dataset) |
| `penguins.csv` | [Palmer Penguins](https://github.com/allisonhorst/palmerpenguins) (CC0) |
